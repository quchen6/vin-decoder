== VIN Decoder

A quick overview of some features:

* VIN Decoder

* TCO (True Cost of Ownership) Estimator

* AngularJS front end

* Localization/I18n

* Postgres arrays for storing Edmunds' Style Ids

* Postgres JSON for storing additional Edmunds attributes

* RSpec for unit testing ruby code

* Jasmine for testing JS code

* Bootstrap 3
